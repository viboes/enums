Function template make_range

boost::enums::make_range — function to generate an enum range.

Synopsis

// In header: <boost/enums/containers/enum_range.hpp>


template<typename T> enum_range< T > make_range();

Description

make_range allows treating enums as a model of the RandomAccessRange Concept. T is a model of the Enumeration Concept. an enum range from T'first to T'last inclusive.
