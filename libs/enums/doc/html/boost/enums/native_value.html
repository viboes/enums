Function template native_value

boost::enums::native_value — gets the native enum value.

Synopsis

// In header: <boost/enums/scoped/native_value.hpp>


template<typename EC> native_type< EC >::type native_value(EC e);

Description

This function is useful to make programs portable when the scoped enum emulation can not be use where native enums can.

Template Parameters:

- EC: the scoped enum type.

Parameters:

- e: the scoped enum.

Returns: the native enum value.

Exception Safety: no-throw guarantee
