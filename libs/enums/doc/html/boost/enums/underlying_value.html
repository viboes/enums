Function template underlying_value

boost::enums::underlying_value — gets the underlying value.

Synopsis

// In header: <boost/enums/scoped/underlying_value.hpp>


template<typename EC> underlying_type< EC >::type underlying_value(EC e);

Description

This function is useful to when working with scoped enum classes which doens't implicitly convert to the underlying type.

Template Parameters:

- EC: the scoped enum type.

Parameters:

- e: the scoped enum.

Returns: the underlying value.

Exception Safety: no-throw guarantee
