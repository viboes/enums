Struct template scoping_type

boost::enums::scoping_type

Synopsis

// In header: <boost/enums/scoped/scoping_type.hpp>

template<typename EC_type> 
struct scoping_type {
  // types
  typedef meta::scoping_type< EC_type >::type type;
};

Description

Get the wrapping class of an enum when emulation is used or the enum class itself when available.

Note(s): This meta-function must be specialized for each enum class.

Requires: EC must be an enum type or the emulation of a scoped enum.

scoping_type public types

  1. typedef meta::scoping_type< EC_type >::type type;

    Depending on whether the compiler supports scoped enums or not the nested type must be the same type or the emulated one.
