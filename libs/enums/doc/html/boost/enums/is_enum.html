Struct template is_enum

boost::enums::is_enum

Synopsis

// In header: <boost/enums/scoped/is_enum.hpp>

template<typename T> 
struct is_enum : public bool_type {
};

Description

meta-function checking if his parameter is a built-in or emulated enum, that is can be used as an emulated enum.The member value states whether T can be used as an enum. When scoped enums are not supported it is defined as std::is_enum<T>::value or is_emulated_enum<T>::value. Otherwise is defined as std::is_enum<T>::value.
