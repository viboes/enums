Class template enum_range

boost::enums::enum_range

Synopsis

// In header: <boost/enums/containers/enum_range.hpp>

template<typename T> 
class enum_range {
public:
  // construct/copy/destruct
  enum_range();
  enum_range(T, T);
};

Description

Template Parameters:

- T: set's element ordinal enum.

Requires: T must be a model of OrdinalEnum.

enum_range is a model of the RandomAccessRange Concept associated to the enumeration T.

enum_range public construct/copy/destruct

  1. enum_range();
    builds a enum range 
  2. enum_range(T first, T last);
    builds a enum sub-range
