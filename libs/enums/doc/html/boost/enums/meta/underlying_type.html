Struct template underlying_type

boost::enums::meta::underlying_type — customization-point for underlying type metafunction

Synopsis

// In header: <boost/enums/scoped/underlying_type.hpp>

template<typename EC> 
struct underlying_type {
  // types
  typedef see_below type;
};

Description

Requires EC must be an enum type or the emulation of a scoped enum.

underlying_type public types

  1. typedef see_below type;

    The member typedef type name the underlying type of T. When scoped enums are emulated it is defined as typename EC::underlying_type. Otherwise is defined as std::underlying_type<EC>::type. This meta-function must be specialized for the compilers providing scoped enums but don't providing the std::underlying_type meta-function.
