Struct template scoping_type

boost::enums::meta::scoping_type — meta-function to be specialized for each emulated enum class.

Synopsis

// In header: <boost/enums/scoped/scoping_type.hpp>

template<typename EC_type> 
struct scoping_type {
  // types
  typedef EC_type type;
};

Description

Requires EC must be an enum type or the emulation of a scoped enum.

scoping_type public types

  1. typedef EC_type type;

    By default this metafunction defines it as if scoped enums where supported.
