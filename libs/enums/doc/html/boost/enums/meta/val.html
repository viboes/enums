Struct template val

boost::enums::meta::val — meta-function used to associate the position to an element of an enumeration.

Synopsis

// In header: <boost/enums/ordinal/val.hpp>

template<typename EC, std::size_t I> 
struct val {

  // public data members
  constexpr native_type< EC >::type value;
};

Description

Note(s): This meta-function must be specialized for each position of the enumeration, starting from 0 to the predecessor of the size of the enumeration .

val public public data members

  1. constexpr native_type< EC >::type value;

    The nested value to be defined for each scoped enum specialization.
