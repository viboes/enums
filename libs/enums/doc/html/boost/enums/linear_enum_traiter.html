Struct template linear_enum_traiter

boost::enums::linear_enum_traiter — Helper class used as enum_trait<> for enumerations having a linear progression.

Synopsis

// In header: <boost/enums/ordinal/linear_traiter.hpp>

template<typename EC> 
struct linear_enum_traiter {

  // public static functions
  static constexpr std::size_t pos(EC);
  static constexpr EC val(std::size_t);
};

Description

linear_enum_traiter public static functions

  1. static constexpr std::size_t pos(EC e);
    pos specialization.

    the returned value is calculated from the underlying value, the first_value and the step, following this formula (ut-first_value)/step 

  2. static constexpr EC val(std::size_t p);
    val specialization.

    Returns: the returned value is calculated from the position p, the first value and the step, following this formula p*step+first_value
