Function template make_range

boost::enums::make_range — function to generate an enum sub-range.

Synopsis

// In header: <boost/enums/containers/enum_range.hpp>


template<typename T> enum_range< T > make_range(T first, T last);

Description

make_range allows treating enums as a model of the RandomAccessRange Concept. It should be noted that the first and last parameters denoted a closed range.

Requires: T is a model of the Enumeration Concept.

Parameters:

- first: first element of the range.

- last: last element of the range.

Returns: an enum range from first to last inclusive.
