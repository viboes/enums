Function template default_value

boost::enums::default_value — builds a enum class with the default value.

Synopsis

// In header: <boost/enums/scoped/default_value.hpp>


template<typename EC> EC default_value();

Description

This function is usefult to make programs portable when the scoped enum emulation doesn't defines a default constructor.

Template Parameters:

- EC: the scoped enum type.

Returns: the default value as if the default constructor of EC was defined.

Exception Safety: no-throw guarantee

Note(s): The use of EC() will let the underlying storage uninitialized when constructors are not defined by the emulation.
