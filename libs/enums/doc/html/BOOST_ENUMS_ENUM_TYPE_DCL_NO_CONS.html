Macro BOOST_ENUMS_ENUM_TYPE_DCL_NO_CONS

BOOST_ENUMS_ENUM_TYPE_DCL_NO_CONS — Generates all the needed definition associated to an ordinal scoped enum type with string conversions and without constructors.

Synopsis

// In header: <boost/enums/pp/enum_declaration.hpp>

BOOST_ENUMS_ENUM_TYPE_DCL_NO_CONS(NS_EC, UT, EL, TRAITER)

Description

Parameters:

- NS_EC: the NAMESPACES_CLASS sequence.

- UT: the underlying type.

- EL: the ENUMERATOR_LIST sequence.

- TRAITER: the enum traiter template class.

Result: 

BOOST_ENUM_NS_TYPE_START(NS_EC, UT)
{
  BOOST_ENUMS_ENUMERATOR_LIST_GENERATE(EL)
}
BOOST_ENUM_NS_TYPE_NO_CONS_END(NS_EC, UT)
BOOST_ENUMS_ENUM_DCL_SPE(NS_EC, EL, TRAITER)
