Macro BOOST_ENUMS_ENUM_DCL_SIZE_SPE

BOOST_ENUMS_ENUM_DCL_SIZE_SPE — Generates a boost::enums::meta::size specialization.

Synopsis

// In header: <boost/enums/pp/enum_declaration.hpp>

BOOST_ENUMS_ENUM_DCL_SIZE_SPE(NS_EC, EL)

Description

Remark(s): This macro is presented here for exposition only reasons and is not part of the interface.

Parameters:

- NS_EC: the NAMESPACES_CLASS sequence.

- EL: the ENUMERATOR_LIST sequence.

Result: 

 template <>
 struct size<BOOST_ENUMS_NAMESPACES_CLASS_QNAME(NS_EC)>
 {
 static const std::size_t value = BOOST_PP_SEQ_SIZE(EL);
 };
