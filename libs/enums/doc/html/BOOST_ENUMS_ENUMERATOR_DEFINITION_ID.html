Macro BOOST_ENUMS_ENUMERATOR_DEFINITION_ID

BOOST_ENUMS_ENUMERATOR_DEFINITION_ID — This macro is used to get the identifier from a ENUMERATOR_DEFINITION.

Synopsis

// In header: <boost/enums/pp/enumerator_definition.hpp>

BOOST_ENUMS_ENUMERATOR_DEFINITION_ID(ED)

Description

Parameters:

- ED: the ENUMERATOR_DEFINITION.

Result: the first element of ED.
