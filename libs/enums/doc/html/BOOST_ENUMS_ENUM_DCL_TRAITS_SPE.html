Macro BOOST_ENUMS_ENUM_DCL_TRAITS_SPE

BOOST_ENUMS_ENUM_DCL_TRAITS_SPE — Generates the enum_trait specialization.

Synopsis

// In header: <boost/enums/pp/enum_declaration.hpp>

BOOST_ENUMS_ENUM_DCL_TRAITS_SPE(NS_EC, TRAITER)

Description

Remark(s): This macro is presented here for exposition only reasons and is not part of the interface.

Parameters:

- NS_EC: the NAMESPACES_CLASS sequence.

- TRAITER: the enum traiter template class.

Result: 

template <>
struct enum_traits<BOOST_ENUMS_NAMESPACES_CLASS_QNAME(NS_EC)>
  : TRAITER<BOOST_ENUMS_NAMESPACES_CLASS_QNAME(NS_EC)>
{};
