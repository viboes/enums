Macro BOOST_ENUMS_ENUMERATOR_DEFINITION_CPP_ENUMERATOR_DEFINITION

BOOST_ENUMS_ENUMERATOR_DEFINITION_CPP_ENUMERATOR_DEFINITION — generates a enumerator_definition from a ENUMERATOR_DEFINITION.

Synopsis

// In header: <boost/enums/pp/enumerator_definition.hpp>

BOOST_ENUMS_ENUMERATOR_DEFINITION_CPP_ENUMERATOR_DEFINITION(ED)

Description

Parameters:

- ED: the ENUMERATOR_DEFINITION.

- DEFAULT: the default value when there is no second element.

Result: 

ID(ED) [= VAL(ED)]

Note(s): The optional part is generated if there is a second element.
